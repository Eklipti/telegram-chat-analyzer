# Возможности и аналитика

В этом документе подробно описаны все метрики и возможности, которые предоставляет Telegram Chat Analyzer.

## Основной процесс
Весь процесс разделен на логические шаги:
1.  **Нормализация** (`step1`): Очистка данных, нормализация времени, обогащение категорий медиа.
2.  **Агрегация** (`step2`): Расчет базовой статистики (топы, динамика).
3.  **Социальный граф** (`step3.5`): Углубленная статистика взаимодействий и контентный анализ.
4.  **Отчетность** (`step3-4`): Генерация финальных отчетов в HTML и Excel.

Команды запуска для каждого шага (и полного конвейера) описаны в [`docs/tutorials.md`](tutorials.md).

## Форматы отчетов
1.  **Desktop HTML:** Интерактивный однофайловый отчет с графиками на JS/Canvas.
2.  **Mobile HTML:** Адаптивная версия, оптимизированная *(в будущем)* для мобильных устройств.
3.  **Excel (.xlsx):** Данные для сводных таблиц, микрографики и условное форматирование.

## Подробный список метрик

### 1. Взаимодействия (Социальный граф)
* **Индекс цитируемости:** На чьи сообщения чаще всего отвечают.
* **Матрица упоминаний:** Кого чаще всего упоминают через @username.
* **Карта симпатий:** Кто кому чаще отвечает (матрица взаимодействий пар).

### 2. Контентный анализ
* **Любители голосовых:** Суммарная длительность голосовых сообщений по авторам.
* **Внешние ссылки:** Анализ наиболее часто упоминаемых доменов.
* **Крикуны:** Процент сообщений, написанных с использованием CAPS LOCK.
* **Оформители:** Использование форматирования текста.
* **Словарный запас (MATTR):** Лексическое разнообразие с использованием лемматизации (pymorphy3).

### 3. Поведенческие паттерны
* **Скорость реакции:** Медианное время ответа на сообщения.
* **"Совы" против "Жаворонков":** Категоризация по времени суток.
* **Индекс "Самоцензуры":** Частота редактирования сообщений после отправки.
